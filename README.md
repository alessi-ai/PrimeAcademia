# Assistant Médical IA

Cette application est une démonstration d'un assistant médical IA conçu pour fournir des informations médicales générales et éducatives.

## Fonctionnalités

- **Chatbot IA:** Interagissez avec un assistant IA pour obtenir des informations médicales.
- **Authentification simulée:** Le système peut être configuré pour vérifier si un utilisateur est authentifié avant de répondre.
- **Logique du prompt système:** L'IA suit un ensemble de règles strictes, notamment en ne fournissant pas de diagnostics ou de prescriptions.

## Démarrage

1. Ouvrez `index.html` dans votre navigateur.
2. Posez une question dans le champ de saisie et cliquez sur "Envoyer".
